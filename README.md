# FunnySkAddon

### O Pluginie: Jest to plugin dodający do minecrafta wsparcie pluginu funnyguilds dla skript
### Twórcy: McKoxu, LloydPL
### Dokumentacja: https://mlgroup.pl/funnyskaddon/home
